Vim
###

Basic configuration
===================

* Show line numbers:

.. code-block:: bash

    :set number

* Enable syntax highlighting:

.. code-block:: bash

    :syntax on

* Enable and disable line wrapping:

.. code-block:: bash

    :set wrap
    :set nowrap

Check spelling
==============

Go to the following or previous error

`[s` `]s`

Show recomendations

`=z`
