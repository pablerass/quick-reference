Vim
###

Basic configuration
===================

* Show line numbers:

.. code-block:: bash

    :set number

* Enable syntax highlighting:

.. code-block:: bash

    :syntax on

* Enable and disable line wrapping:

.. code-block:: bash

    :set wrap
    :set nowrap

* Use spaces instead of tabs

.. code-block:: bash

    :set expandtab

Check spelling
==============

* ``[s`` ``]s`` - Go to the following or previous error.

* ``=z`` - Show recomendations
